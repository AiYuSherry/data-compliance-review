# 数据合规智能审查系统 - Web界面

让不懂计算机的法务/合规人员也能轻松使用的数据合规审查工具。

## 快速开始

### macOS/Linux
1. 打开终端，进入本文件夹
2. 运行：`./start.sh`
3. 浏览器会自动打开 `http://127.0.0.1:5000`

### Windows
1. 双击运行 `start.bat`
2. 浏览器会自动打开 `http://127.0.0.1:5000`

## 使用步骤

1. **输入文档名称**（例如：某App隐私政策V1.0）
2. **上传文件**或**粘贴文本**
3. 点击**开始审查**
4. 等待进度条完成（约10-30秒）
5. 查看审查结果

## 输出内容

- 风险等级统计（高风险/中风险/建议优化）
- 详细风险分析（原因、法规依据、修改建议）
- 证据片段引用
- 整改任务清单（可下载）
- 完整JSON报告（可下载）

## 功能特点

- ✅ 拖拽上传文件
- ✅ 直接粘贴文本
- ✅ 实时进度显示（10个检查点逐一展示）
- ✅ 可视化风险统计
- ✅ 一键下载报告
- ✅ 本地运行，数据不上传云端

## 注意事项

- 首次启动会自动安装依赖（Flask等），请保持网络连接
- 支持的文件格式：.txt, .md, .doc, .docx
- 审查结果仅供参考，不替代最终法律判断

## 技术说明

- 后端：Python + Flask
- 前端：HTML + TailwindCSS
- 审查引擎：基于规则的静态分析
